The three timer types
Fixed

One timer for the whole block. Headcount doesn't matter. Use it for rituals, breaks, and deep dives.

Everyone

You set the total; it splits between everyone. More people → less time each, but the block stays the same length.

Per person

You set the time each person gets. More people → the block gets longer, but everyone keeps their full slot.

Everyone & Per person look identical on screen — they're just opposite math. Pick the number you want to lock in.
