# EO Forum Timer

A mobile-first, full-screen timer for running EO Forum meetings.

- Build a schedule of activities (fixed blocks, or time split per person).
- Per-person activities show one vertical bar per person; the timekeeper starts each person manually.
- Full-screen color bars, overtime count-up, and live "ahead/behind schedule" pace.
- The whole schedule is saved in the URL — copy the link to share or reload it. It also remembers your last setup locally.
- "Reset to standard" restores the standard EO Forum agenda.

Single self-contained `index.html` — no build step, no backend.
